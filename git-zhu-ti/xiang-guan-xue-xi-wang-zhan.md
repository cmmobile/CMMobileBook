## GitLab + SSH key

* [在 GitLab 上新增 SSH key](https://sinyilin.github.io/git/20191024/1014042378/)

可能新電腦第一次用SourceTree會無法使用ssh，會一直轉轉轉不停，原因是缺少\`ecdsa key fingerprint\`，請直接在終端機下指令下載專案，往後都可以直接在SourceTree用ssh。

```bash
git clone 你的專案ssh連結
```
